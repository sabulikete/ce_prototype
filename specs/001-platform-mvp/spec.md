## Primary User Scenarios

(See [Requirements](requirements.md) for detailed user stories)

### 1. Public Access
Guests can view public announcements and events without logging in.

### 2. Member Access
Members log in to view private content, billing statements, and tickets.

### 3. Event Check-in
Staff scan member QR codes to validate entry to events.

### 4. Admin Management
Admins manage content, users, and billing uploads via the dashboard.
